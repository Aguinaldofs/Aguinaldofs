## About

Backend Developer with experience in **Java**, **Spring Boot**, **OSGi**, **REST APIs** and **microservices development**.

I have worked with API exposure layers, backend services and automation in both **virtualized** and **containerized environments**, using tools such as **Docker**, **Kubernetes**, **Jenkins**, **Git/Gerrit**, **Jira**, **JCAT/Cucumber** and **Postman**.

I hold a **Bachelor's degree in Computer Engineering** and a **Postgraduate degree in Cloud Computing and Mobile Devices**. I also have academic and practical experience with **AWS services**, cloud computing concepts and backend architecture.
